# Mindo - Obsidian 思维导图插件

Mindo 是一个功能强大的 Obsidian 思维导图插件，帮助你可视化思考、整理知识和创意。

## 功能特点

### 核心功能
- 📊 交互式思维导图编辑
- 📝 Markdown 双向同步
- 🖼️ 图片节点支持
- 🔗 节点连接与关系管理
- 🎨 自定义节点样式和颜色
- 🔍 节点搜索功能
- 📋 节点拖拽与重排序
- 💾 自动保存功能

### 高级功能
- 🤖 AI 辅助内容生成（头脑风暴、内容深化）
- 🎯 智能配色方案
- 📦 节点分组管理
- 🔄 撤销/重做操作
- 📷 思维导图导出为图片
- 🌓 深色/浅色模式支持

## 安装方法

1. 在 Obsidian 中打开 "设置" → "插件"
2. 点击 "社区插件" 选项卡
3. 搜索 "Mindo"
4. 点击 "安装" 按钮
5. 安装完成后，点击 "启用" 按钮

## 基本使用

### 创建思维导图

1. 在 Obsidian 中创建一个新文件
2. 将文件类型设置为 "Mindo 思维导图"
3. 点击 "打开思维导图" 按钮进入编辑模式

### 基本操作

#### 添加节点
- **双击画布**：在空白处双击添加新节点
- **拖拽节点**：点击节点并拖动到新位置
- **连接节点**：从一个节点的连接点拖拽到另一个节点

#### 编辑节点
- **双击节点**：进入编辑模式
- **编辑标题**：在节点顶部输入标题
- **编辑内容**：在节点内容区域输入详细信息
- **添加图片**：直接拖放图片到画布，或使用右键菜单添加

#### 节点样式
- **右键点击节点**：打开节点菜单
- **选择颜色**：在菜单中选择节点颜色
- **添加图标**：在菜单中选择节点图标

### Markdown 同步

Mindo 支持与 Markdown 双向同步：

1. 在思维导图编辑模式下，点击顶部的 "Markdown" 按钮切换到 Markdown 视图
2. 在 Markdown 视图中编辑内容，格式如下：
   ```markdown
   # 中心主题
   
   ## 子主题 1
   内容描述
   
   ## 子主题 2
   ![[图片路径]]
   ```
3. 切换回思维导图视图，内容会自动同步

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Y` | 重做 |
| `Ctrl/Cmd + C` | 复制选中节点 |
| `Ctrl/Cmd + V` | 粘贴节点 |
| `Delete` | 删除选中节点 |
| `Ctrl/Cmd + 滚轮` | 缩放画布（已优化为直接滚轮缩放） |
| `Shift + 点击` | 多选节点 |

## AI 功能

### 头脑风暴
1. 右键点击节点
2. 选择 "头脑风暴"
3. AI 会基于节点内容生成相关的子节点

### 内容深化
1. 右键点击节点
2. 选择 "内容深化"
3. AI 会为节点生成更详细的内容

## 导出功能

### 导出为图片
1. 点击顶部工具栏的 "导出" 按钮
2. 选择 "导出为图片"
3. 选择保存位置和图片格式

### 导出为 Markdown
1. 点击顶部工具栏的 "导出" 按钮
2. 选择 "导出为 Markdown"
3. 选择保存位置

## 常见问题

### 1. 图片不显示
- 确保图片路径正确
- 检查 Obsidian 的资源路径设置
- 尝试重新拖放图片

### 2. 连接线抖动
- 编辑节点时连接线可能会有轻微调整，这是正常现象
- 编辑完成后连接线会稳定

### 3. 性能问题
- 对于大型思维导图，建议使用分组功能
- 定期保存思维导图以避免数据丢失

## 注意事项

- Mindo 会自动保存你的思维导图
- 建议定期备份思维导图文件
- 对于大型思维导图，可能会有一定的性能影响
- 图片节点会占用更多内存，建议合理使用
