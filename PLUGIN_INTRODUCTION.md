# Mindo 插件介绍

## 插件概述

### 插件名称
Mindo

### 插件类型
Obsidian 社区插件

### 插件版本
v1.0.0

### 插件简介
Mindo 是一个功能强大的 Obsidian 思维导图插件，帮助用户可视化思考、整理知识和创意，支持 Markdown 双向同步和 AI 辅助功能。

## 核心功能

### 思维导图编辑
- 交互式节点编辑
- 节点拖拽与重排序
- 节点连接与关系管理
- 自定义节点样式

### Markdown 同步
- 双向同步功能
- 支持 Obsidian 双链格式
- 图片节点自动转换

### 图片支持
- 拖放图片添加
- 图片节点显示
- 图片路径自动解析

### AI 辅助功能
- 头脑风暴生成
- 内容深化扩展
- 智能创意生成

## 技术特点

### 前端技术
- React 框架
- TypeScript 类型系统
- 响应式设计

### 性能优化
- 防抖和节流处理
- 虚拟滚动
- 高效渲染

### 数据管理
- 自动保存机制
- 历史记录管理
- 撤销/重做功能

## 安装方法

### 社区插件安装
1. 打开 Obsidian 设置
2. 进入社区插件选项卡
3. 搜索 Mindo
4. 点击安装并启用

### 手动安装
1. 下载插件文件
2. 放置到 Obsidian 插件目录
3. 重启 Obsidian 并启用

## 常见问题

### 图片不显示
- 检查图片路径是否正确
- 确认 Obsidian 资源设置
- 尝试重新添加图片

### 性能问题
- 对于大型思维导图，建议使用分组
- 定期保存避免数据丢失
- 关闭不必要的功能

### 同步问题
- 确保 Markdown 格式正确
- 检查文件权限
- 尝试重新同步
