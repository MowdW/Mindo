# Mindo 思维导图插件使用指南

## 快速开始

### 安装插件
1. 在 Obsidian 中打开 "设置" → "插件"
2. 点击 "社区插件" 选项卡
3. 搜索 "Mindo"
4. 点击 "安装" 按钮
5. 安装完成后，点击 "启用" 按钮

### 创建第一个思维导图
1. 在 Obsidian 中创建一个新文件
2. 将文件类型设置为 "Mindo 思维导图"
3. 点击 "打开思维导图" 按钮进入编辑模式
4. 双击画布添加第一个节点
5. 输入节点标题，按 Enter 确认

## 基本操作

### 添加节点
- **双击画布**：在空白处双击添加新节点
- **快捷键**：
  - `Enter`：在当前节点后创建同级节点
  - `Tab`：创建当前节点的子节点
- **右键菜单**：右键点击画布，选择 "添加节点"

### 编辑节点
- **双击节点**：进入编辑模式
- **编辑标题**：在节点顶部输入标题
- **编辑内容**：在节点内容区域输入详细信息
- **完成编辑**：按 Enter 或点击节点外部

### 连接节点
1. 鼠标悬停在一个节点上，会出现四个连接点
2. 点击并拖拽其中一个连接点
3. 释放鼠标到另一个节点上
4. 两个节点之间会创建连接线

### 移动节点
- **拖拽节点**：点击节点标题区域并拖动到新位置
- **微调位置**：选中节点后，使用方向键微调位置

### 删除节点
- **选中节点**：点击节点
- **按 Delete 键**：删除选中的节点
- **右键菜单**：右键点击节点，选择 "删除节点"

## 节点样式

### 节点颜色
1. 右键点击节点
2. 选择 "节点颜色"
3. 从颜色面板中选择一种颜色

### 节点图标
1. 右键点击节点
2. 选择 "添加图标"
3. 从图标列表中选择一个图标
4. 选择图标位置（角落或内联）

### 节点大小
1. 选中节点
2. 拖动节点右下角的调整手柄
3. 调整到合适的大小

## 图片节点

### 添加图片
- **拖放图片**：直接将图片文件拖放到画布上
- **复制粘贴**：复制图片后，在画布上粘贴
- **右键菜单**：右键点击画布，选择 "添加图片"

### 编辑图片节点
- **双击图片节点**：进入编辑模式
- **修改标题**：编辑图片节点的标题（默认为图片文件名）
- **调整大小**：拖动图片节点的调整手柄

### 图片格式支持
- 支持常见图片格式：JPG、PNG、GIF、WebP
- 支持 Obsidian 附件路径：`![[附件/图片.png]]`

## Markdown 同步

### 切换到 Markdown 视图
1. 点击顶部工具栏的 "Markdown" 按钮
2. 进入 Markdown 编辑模式

### Markdown 格式
```markdown
# 中心主题

## 子主题 1
内容描述

## 子主题 2
! [[附件/图片.png]]
```

### 同步回思维导图
1. 在 Markdown 视图中编辑内容
2. 点击顶部工具栏的 "思维导图" 按钮
3. 内容会自动同步到思维导图

## 高级功能

### 节点分组
1. 按住 Shift 键选择多个节点
2. 右键点击选中的节点
3. 选择 "创建分组"
4. 输入分组名称

### AI 辅助功能

#### 头脑风暴
1. 右键点击一个节点
2. 选择 "头脑风暴"
3. AI 会基于节点内容生成相关的子节点
4. 等待生成完成，查看结果

#### 内容深化
1. 右键点击一个节点
2. 选择 "内容深化"
3. AI 会为节点生成更详细的内容
4. 内容会自动添加到节点的内容区域

### 智能配色
1. 右键点击画布
2. 选择 "智能配色方案"
3. 插件会自动为节点分配协调的颜色

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Y` | 重做 |
| `Ctrl/Cmd + C` | 复制选中节点 |
| `Ctrl/Cmd + V` | 粘贴节点 |
| `Delete` | 删除选中节点 |
| `滚轮` | 缩放画布（以鼠标位置为中心） |
| `Shift + 点击` | 多选节点 |
| `Ctrl/Cmd + 点击` | 平移画布 |

## 导出功能

### 导出为图片
1. 点击顶部工具栏的 "导出" 按钮
2. 选择 "导出为图片"
3. 选择保存位置和图片格式
4. 点击 "保存"

### 导出为 Markdown
1. 点击顶部工具栏的 "导出" 按钮
2. 选择 "导出为 Markdown"
3. 选择保存位置
4. 点击 "保存"

## 视图控制

### 缩放控制
- **滚轮**：向前滚动放大，向后滚动缩小
- **缩放按钮**：点击顶部工具栏的缩放按钮
- **重置视图**：点击顶部工具栏的 "重置视图" 按钮

### 视图模式
- **思维导图视图**：默认视图，交互式编辑
- **Markdown 视图**：文本编辑模式

## 常见问题

### 图片不显示
- 确保图片路径正确
- 检查 Obsidian 的资源路径设置
- 尝试重新拖放图片

### 连接线抖动
- 编辑节点时连接线可能会有轻微调整，这是正常现象
- 编辑完成后连接线会稳定

### 性能问题
- 对于大型思维导图，建议使用分组功能
- 定期保存思维导图以避免数据丢失
- 关闭不必要的功能

### 同步问题
- 确保 Markdown 格式正确
- 检查文件权限
- 尝试重新同步
